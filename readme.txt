配置github远程仓库的命令（在SSH秘钥已经和github网站上关联后）
在本地仓库中打开Terminal

1、git init
2、git config --global user.name "github账户名"
3、git config --global user.email "github注册所用邮箱"
4、在github上建立一个同名仓库
5、git remote add origin git@github.com:github账户名/仓库名

然后就可以正常使用了
